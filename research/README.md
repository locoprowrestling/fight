# Research

Files in this directory are **informational background only**.

They are not implementation guides, specifications, roadmaps, backlogs,
acceptance criteria, or sources of truth for the game. They may contain
speculative ideas, generic examples, pseudocode, prompts, and recommendations
that have not been reviewed or adopted for this repository.

Use these files to identify potentially useful practices, then validate each
idea against this game's goals, code, and constraints. Do not implement or
refactor from research alone. Accepted findings must be rewritten as explicit
project decisions in the appropriate authoritative documentation.

Current authority lives in:

- runtime code under `Assets/Scripts/`
- [Documentation/DesignDoc.md](../Documentation/DesignDoc.md)
- [Documentation/KnowledgeBase/BestPractices.md](../Documentation/KnowledgeBase/BestPractices.md)
- [Documentation/KnowledgeBase/Templates.md](../Documentation/KnowledgeBase/Templates.md)
- [Documentation/TestingChecklist.md](../Documentation/TestingChecklist.md)

[Documentation/KnowledgeBase/ResearchNotes.md](../Documentation/KnowledgeBase/ResearchNotes.md).
