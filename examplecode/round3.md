## CRITICAL DISTINCTION FROM PRIOR ROUNDS

Rounds 01–02 analyzed the **standard move library** (weak/strong grapple pools assignable to directional slots).
This round catalogs the **SPECIAL move layer** — a separate, elevated tier that:

- Requires the momentum meter to reach the SPECIAL threshold before becoming executable
- Cannot be assigned to standard directional grapple slots
- Are designated per-wrestler as their signature/finisher
- Represent the highest-damage, highest-visual-impact animations in the game

This is the full move pool from which a wrestler's finisher is selected in the character-creation system.

## CATEGORY 1 — FRONT SPECIALS

**Position:** Attacker standing, facing opponent standing
**Total moves:** ~60 (largest pool in the game)

### Notable System Properties Observed

- Some moves integrate a **pin** directly into the animation (listed as "Pin" variants)
- Some moves trigger **Bloodshed** — damage type flags that cause the opponent to bleed
- Some moves carry a **knockout** chance (separate from standard pin victory)
- Two moves use **special effects** — Fireball (face turns black) and Poison Mist (face turns the color of the mist: blue/red/green/white/black)
- Mandible Claw is noted explicitly as **NOT causing Bloodshed** — contrasted directly with Iron Claw which does; this is a documented mechanic difference between two otherwise similar claw-hold moves
- Hip Toss into Submission is **modular** — the submission component is chosen separately by the player, meaning one finisher slot can reference two moves
