# Powerbomb

- ID: `round03-powerbomb`
- Formation: front standing, roots 0.85 m apart.
- Contacts: defender head between attacker thighs; attacker grips waist.
- Phases: setup 0-33%, lift 33-67%, impact 67-100%.
- Markers: `contact`, `pose-sync` at shoulder height, `impact`.
- Exit: defender face-up with pin window; attacker standing over feet.
- Interruption: setup and early lift.
- Weight variants: heavy-target version required when hip height drifts.
- Fallback: overhead slam / lifted defender.
