# Shared Style Guide

Create a full-body professional wrestling character portrait in a clean,
boldly outlined 2D digital cartoon style. Use confident black linework, flat
colors, restrained cel shading, expressive facial features, and clearly
readable costume details. Show the wrestler standing alone in a strong,
character-specific pose, fully visible from head to boots. Use a transparent
background with no scenery, border, caption, logo, or nameplate. Keep the
silhouette distinct and suitable for a fighting-game character-select screen.

Target output: vertical PNG, approximately 259 x 400 pixels, transparent
background.
